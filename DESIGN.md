# Design system

The site behaves like a sharp factory order board brought to life: black working surfaces, hard white structure, signal red actions, and real food photography.

## Color

- Black: `oklch(0.08 0 0)`
- Surface: `oklch(0.14 0.008 25)`
- White: `oklch(0.985 0 0)`
- Factory red: `oklch(0.58 0.23 27)`
- Safety yellow: `oklch(0.83 0.17 85)`

## Typography

Heavy condensed system display type for headings and a familiar sans-serif for fast menu reading.

## Interaction

Rectangular controls, strong focus rings, 44px minimum targets, a compact mobile navigation, and reduced-motion support.
